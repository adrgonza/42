# 42 — Adrián González Serrano

> Every project from my 42 Madrid · Lausanne cursus (2022 – 2026),
> in one place. A single repository to see the full arc.

[42](https://www.42network.org/) is a peer-to-peer computer science program
with no teachers and no lectures. You learn by building, by being reviewed
by your peers, and by failing in public until your code holds. This repo
is the full arc of that journey.
